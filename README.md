# react-native-audienzz

React Native implementation of the AppNexus (Xandr) ads

## Installation

```sh
npm install react-native-audienzz
```

## Usage

```js
import Audienzz from "react-native-audienzz";

// ...

const result = await Audienzz.multiply(3, 7);
```
